# 🎵 2TUFF.wav - Play your music on PSP devices

[![](https://img.shields.io/badge/Download-2TUFF.wav-blue)](https://github.com/Udantha4748/2TUFF.wav)

2TUFF.wav serves as a music player for the PlayStation Portable. It allows users to listen to audio files directly on their hardware. The application organizes your music library and provides playback controls for different file formats. This software works on custom firmware. 

## 📥 Getting Started

You need the correct files to use this player. Visit the [project page](https://github.com/Udantha4748/2TUFF.wav) to download the software. Select the latest release version on the right side of the screen. Look for the file ending in .zip. Save this file to your computer.

## ⚙️ System Requirements

Before you install 2TUFF.wav, check that your device meets these needs:

* A PlayStation Portable console.
* Custom firmware already installed on the handheld.
* A Memory Stick Pro Duo card with available storage.
* A computer running Windows 10 or 11.
* A USB cable to connect your console to the computer.

## 🛠️ Installation Steps

Follow these instructions to move the music player to your device. 

1. Connect your PSP to the Windows computer using a USB cable.
2. Open the File Explorer on your computer.
3. Locate the PSP drive. It usually appears under This PC.
4. Open the PSP folder on the root of your memory card.
5. Create a folder named GAME if one does not exist.
6. Open the .zip file you downloaded earlier.
7. Extract the contents into the PSP/GAME folder.
8. Ensure the application folder contains an EBOOT.PBP file.
9. Disconnect the USB cable from the computer safely.

## 🎧 How to Listen to Music

After the installation, you can launch the player.

1. Navigate to the Game menu on your PSP home screen.
2. Look for the 2TUFF.wav icon in the Memory Stick section.
3. Press the X button to start the program.
4. Use the D-pad to browse your music folders.
5. Press the X button to select a track.
6. Use the shoulder buttons to adjust volume levels.

## 📁 Supported Audio Formats

This player handles common audio files to ensure compatibility.

* WAV: Provides high-quality playback for your audio needs.
* MP3: Offers a balance between file size and sound clarity.

Keep your file names short to avoid errors during loading. Avoid using special characters in file names. Create folders to sort your music by artist or album. This helps the software load your library faster.

## 💡 Troubleshooting Common Issues

If the music player fails to load, check these points:

* Check that you copied the files to the correct folder. 
* Confirm that your custom firmware supports homebrew applications.
* Verify the file structure. The EBOOT.PBP file must reside in its own subfolder within the GAME folder.
* Restart the PSP device if the icon does not appear in the menu.
* Check the Memory Stick for physical damage.

## ❓ Frequently Asked Questions

Does this software harm my device?
No, 2TUFF.wav runs as a standard homebrew application. It does not modify system files on the flash memory of your PSP.

Can I put music anywhere on the stick?
The player searches for files based on the directory structure you define. Use a Music folder at the root of your memory stick for easier access.

Why does my sound crackle?
High-bitrate files may cause playback issues on older hardware. Lower the bitrate of your files to 128kbps for smooth performance.

Does this work on all PSP models?
The software supports 1000, 2000, 3000, and Street models. Ensure your console runs reliable custom firmware.
